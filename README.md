# IMU-test

スマホ搭載の IMU を使用して開始地点との位置の差分を計算・軌跡を表示します。

計測時はスマホをなるべく寝かせず、進行方向にまっすぐ向かせ、あまり動かさないでください。また、歩幅はなるべく一定にし、急な方向転換や段差は避けてください。歩幅が一定ならペースは不問です。

センサーの謎仕様でスクロール中は動きません。
